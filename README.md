## 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a relationship management dashboard built to help you nurture and maintain your most important connections. It allows you to track interaction frequency, set contact goals, and visualize your social habits so that you never lose touch with the people who matter most.

## 🌐 Live Project

https://ph-assignment-07-rouge.vercel.app/

## 🚀 Key Features

Friendship Health Dashboard: View all your friends in a responsive grid with color-coded status indicators (On-Track, Almost Due, or Overdue) based on your last contact date.

Interactive Interaction Logging: Add "Quick Check-ins" (Call, Text, or Video) from any friend's profile to automatically update your global timeline with real-time toast notifications.

Friendship Analytics: A dedicated statistics page featuring a Pie Chart (powered by Recharts) that breaks down your interaction types to show how you connect with others.

## 🛠️ Technologies Used

Framework: React.js / Next.js

Routing: React Router DOM / App Router

Styling: Tailwind CSS

Data Visualization: Recharts

Deployment: Vercel / Netlify

## 📱 Responsiveness

This application is fully responsive and optimized for Mobile, Tablet, and Desktop screen sizes to ensure you can manage your friendships on the go.

## 📂 Project Structure

Home Page: Banner summary and friend status cards.

Detail Page: In-depth friend profiles with relationship goals and interaction buttons.

Timeline: A chronological history of all calls, texts, and video chats with filtering options.

Stats: Visual breakdown of your social activity.
